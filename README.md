# FileTreer - 文件树生成器

一个基于 tkinter 的图形化文件树生成器，用于扫描文件夹并生成文本格式的文件树，适合用于记笔记。

## 功能特性

- 📁 扫描指定文件夹，生成文件树
- 🎨 支持两种输出格式：

  - ASCII 艺术树格式（使用 │├└ 等字符）
  - Markdown 格式（适合笔记）
- ⚙️ 多设置选项：

  - 忽略隐藏文件/文件夹
  - 自定义忽略清单（类似 .gitignore 格式，每行一个）
  - 最大扫描深度限制（默认 4 层）
  - 每层最大文件数限制（默认 15 个）
  - 根目录文件数不限制
- 💾 多种输出方式：

  - 在界面中预览
  - 复制到剪贴板
  - 保存到文件（默认保存到扫描目录）
  
- ⚠️ 提示：当设置无限制时，会弹出确认对话框

## 系统要求

- Python 3.7 或更高版本
- tkinter（Python 内置，通常已包含）

## 安装

1. 克隆或下载此项目
2. 确保已安装 Python 3.7+

## 使用方法

运行主程序：

```bash
python main.py
```

## 使用说明

### 基本流程

1. 在**配置标签页**中，点击"选择文件夹"按钮，选择要扫描的目录
2. 选择输出格式（ASCII 或 Markdown）
3. 根据需要配置设置（见下方详细说明）
4. 点击"生成文件树"按钮
5. 程序会自动切换到**预览标签页**，显示生成的文件树
6. 点击"复制到剪贴板"或"保存到文件"按钮进行输出

### 设置选项说明

#### 忽略隐藏文件/文件夹

- 勾选后，所有以 `.` 开头的文件/文件夹将被忽略

#### 忽略清单

- 使用类似 `.gitignore` 的格式，每行一个模式
- 支持通配符（如 `*.pyc`, `test*`）
- 支持注释行（以 `#` 开头的行会被忽略）
- 默认忽略清单已预填写，可以直接编辑
- 示例：

  ```
  # 忽略测试文件
  test
  *.log
  build/
  ```

#### 最大深度

- 限制扫描的目录层级深度
- 默认值为 4
- 留空表示无限（会弹出确认对话框）

#### 每层最大文件数

- 限制每个目录层级显示的文件/文件夹数量
- 默认值为 15
- 超过限制时，只显示前 N 个，其余用 `...` 表示
- 留空表示无限（会弹出确认对话框）

#### 根目录文件数不限制

- 勾选后，根目录会显示所有文件/文件夹，不受"每层最大文件数"限制
- 子目录仍然受到限制

## 默认忽略清单

程序默认会忽略以下常见的系统文件和目录：

- `.git`, `.gitignore`
- `__pycache__`, `*.pyc`
- `node_modules`
- `.vscode`, `.idea`
- `.DS_Store`

这些已预填写在忽略清单输入框中，可以根据需要编辑或删除。

├── config.py               # 配置管理

## 配置保存

程序会自动保存配置到项目目录下的 `filetreer_config.json` 文件中，下次启动时会自动加载。
